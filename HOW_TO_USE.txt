📦 第一步：安装插件

  1. 打开 IntelliJ IDEA 或 Android Studio
  2. 进入：Settings/Preferences (macOS: ⌘, | Windows: Ctrl+Alt+S)
  3. 选择：Plugins
  4. 点击：齿轮图标 ⚙️ → Install Plugin from Disk...
  5. 选择文件：build/distributions/openincursor-plugin-1.0.0.zip
  6. 点击 OK，然后重启 IDE

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚙️  第二步：配置 Cursor 命令行（如果需要）

  运行环境检查：
  $ ./check-env.sh

  macOS 用户：
  1. 打开 Cursor 编辑器
  2. 按 Cmd+Shift+P
  3. 输入并执行：Shell Command: Install 'cursor' command in PATH
  
  或者：插件会自动尝试使用 'open -a Cursor'（如果 Cursor.app 已安装）

  Windows 用户：
  - 重新安装 Cursor，勾选 "Add to PATH" 选项

  Linux 用户：
  $ sudo ln -s /path/to/cursor /usr/local/bin/cursor

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 第三步：开始使用

  在 IntelliJ IDEA 或 Android Studio 中：
  
  1. 在项目视图中，右键点击项目根目录
  2. 选择 "Open In" → "Cursor"
  3. 项目将在 Cursor 编辑器中打开！

❓ 常见问题

  Q: 点击菜单没反应？
  A: 运行 ./check-env.sh 检查 Cursor 是否正确安装

  Q: 找不到菜单选项？
  A: 确认插件已安装并启用，重启 IDE

  Q: 提示"无法打开 Cursor"？
  A: 查看 INSTALLATION.md 中的故障排除部分
